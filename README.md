# 客户复购预警系统

> 一人公司客户流失风险监控与挽回策略智能推荐工具

---

## Features / 功能特点

| 功能 | 说明 |
|------|------|
| 客户管理 | 添加/编辑客户信息、消费记录完整管理 |
| 风险分级 | 低/中/高三级风险自动识别 |
| 购买记录 | 支持多次购买记录追踪与统计 |
| CSV导入 | 批量导入客户数据，快速初始化 |
| 搜索筛选 | 按风险级别/姓名灵活搜索客户 |
| 挽回策略 | 针对不同风险等级自动生成策略建议 |
| 数据统计 | 总客户数及各风险等级统计卡片 |
| 仪表盘 | 可视化风险分布和客户列表 |

## Installation / 安装

无需安装，直接在浏览器中打开 `index.html` 即可使用。

```bash
# 克隆仓库
git clone https://github.com/yourusername/customer-repurchase-warning.git

cd customer-repurchase-warning
open index.html
```

## Usage / 使用方法

### 基础用法

1. 打开 `index.html`
2. 「添加客户」手动录入，或「导入」Tab 批量CSV导入
3. 仪表盘自动展示风险分布和客户列表
4. 查看底部的「建议挽回策略」获取行动指引
5. 可编辑客户信息或添加新的购买记录

### 风险分级标准

| 等级 | 条件 | 颜色标识 | 挽回建议 |
|------|------|:--------:|----------|
| 低风险 | 近期有购买，复购正常 | 绿色 | 保持服务，定期关怀 |
| 中风险 | 超过60天未购买 | 黄色 | 发送优惠/新品通知 |
| 高风险 | 超过90天未购买或仅购买1次且超过90天 | 红色 | 主动联系，提供专属优惠 |

### CSV导入格式

```csv
姓名,联系方式,首次购买日期,最近购买日期,购买次数,累计金额,备注
张三,13800138000,2024-01-01,2024-03-01,3,5000, VIP客户
李四,13900139000,2024-02-01,2024-02-01,1,1000, 新客户
```

### 挽回策略模板

```markdown
# 低风险客户策略
- 每月发送产品更新通知
- 邀请参与用户调研
- 赠送会员专属福利

# 中风险客户策略
- 发送限时优惠活动
- 推荐相关产品/服务
- 提供老客户专属折扣

# 高风险客户策略
- 专人电话跟进了解原因
- 提供专属挽回礼包
- 考虑服务升级方案
```
