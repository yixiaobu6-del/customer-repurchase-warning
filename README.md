# 客户复购预警系统

客户突然消失了？等到流失才发现太晚了。
**预测客户流失风险，提前挽回。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## 不再"被动等待"

- **流失预测**：指数衰减模型，30天半衰期科学预测
- **客户画像**：Chart.js雷达图，多维度呈现客户特征
- **RFM+CLV双模型**：复购价值+生命周期价值联合分析
- **预警标记**：高风险客户自动标红，优先跟进

## 怎么用

打开 `index.html` → 添加客户数据 → 查看预警列表
